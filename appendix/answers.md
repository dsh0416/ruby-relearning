# 参考答案

参考答案不唯一，跑得起来就是好答案，跑不起来就是错答案。

## 局部变量

创建两个变量 `a` 和 `b`，给它们赋两个整数，计算并打印它们的和。

```ruby
  a = 1
  b = 1
  puts a + b
```

创建两个变量 `a` 和 `b`，给它们赋两个整数，计算并打印它们的积。

```ruby
  a = 1
  b = 1
  puts a * b
```

创建三个变量 `a` 和 `b` 和 `c`，给 `a` 和 `b` 赋两个整数，计算它们的和，将它们的结果写入 `c` 变量中。

```ruby
  a = 1
  b = 1
  c = a + b
```
